# TruthLens

AI-powered fact-checking platform that verifies claims, headlines, URLs, and trending news using real-time evidence retrieval and credibility scoring.

## Features

- AI claim verification
- Trending news fact-checking
- Credibility score
- Dashboard with saved checks
- Dark / Light mode
- Supabase authentication
- Responsive design

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Supabase
- OpenRouter AI
- Vercel

## Local Setup

```bash
npm install
npm run dev
